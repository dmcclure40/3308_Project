Welcome to LeapArm’s documentation!¶

This page documents the class and members of the leapConnect.py file. This file is run to collect hand inputs from the user and output results to a text file.

To run this file, connect your Leap Motion device and run ‘python leapConnect.py’ in the terminal. Data will both be displayed in the console and update the handVariables.txt file for every frame.

Github Repository: https://github.com/dmcclure40/3308_Project

Contents:

class leapConnect.Listener¶

Class which listens for device and hand changes. Listener updates once every .1 second.

on_connect(controller)¶

Confirm connection to Leap Motion Controller

on_device_failure(controller)¶

Detect Leap Motion controller failure

on_disconnect(controller)¶

Confirm disconnection from Leap Motion Controller

on_exit(controller)¶

Pressing ‘enter’ will exit script

on_frame(controller)¶

Get the most recent frame and report some basic information about hand position and gripper open/close
